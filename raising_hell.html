Raising Hell

Explore a long forgotten island to discover the secrets it holds, but beware, you are not the only one who is lurking. Raising Hell is a first person survival horror game set on a remote island in the 1600s.

Raising Hell was my first venture into horror, my favorite genre. To create this game, I worked in a group of seven using Unreal Engine 4, using Blueprints for our visual scripting.

The main tasks I worked on throughout the process included working on the enemy AI to track the player along with the checkpoint system which is represented by torches that light up when the player is in range. For the torches, I created a fire particle effect within Unreal, the progression can be seen in the following images.

Along with this, I was also tasked with other UI elements such as creating the main and pause menus as well as the credits page. Screenshots of the game are shown below.
